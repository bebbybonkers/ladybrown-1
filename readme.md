# Ruiguan Drivecode
## Default Controls
To turn off a subsystem, such as the intake, press the **same button** again \
**Intake on** - L1 \
**Outtake on** - L2 \
**Matchloader** - Up \
**Outtake (Long Goal)** - R1 \
**Trapdoor (Midgoal Scoring)** - X \
**Descore Wing** - R2 

## Ports
Ports can be updated from **objects.cpp** easily all in one file, rather than having to go to a new file for each port definition.
